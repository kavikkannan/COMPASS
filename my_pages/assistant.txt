# ASSISTANT PAGE - Detailed Description

## Page Overview
The Assistant page is the main AI-powered legal assistance interface of the LexAI application. It provides users with an intelligent chat interface for legal queries, document analysis, and AI-powered legal guidance specifically tailored for Indian law.

## Layout Structure
- **Two-column responsive layout**
- **Left Column**: Main chat interface (flexible width)
- **Right Column**: Sidebar with calendar widget and quick actions (fixed 384px width on desktop)
- **Background**: Gradient from gray-50 to blue-50
- **Full height**: Uses h-screen for complete viewport coverage

## Header Section
- **Title**: "LexAI Assistant" (large, bold text)
- **Subtitle**: "AI-powered legal assistance with document analysis and expert guidance."
- **Positioning**: Top of main content area with margin bottom

## Main Chat Interface (Left Column)

### LexAI Workflows Section
Located at the top of the chat interface with a distinctive header:
- **Header**: "LexAI Workflows" with sparkles icon
- **Four Workflow Buttons** in a 2x2 grid (responsive to 4x1 on mobile):
  1. **Summarize** (Blue theme)
     - Icon: Template icon
     - Action: "Please summarize the key points of my uploaded document."
  2. **Citations** (Green theme)
     - Icon: Case file icon
     - Action: "Please provide legal citations for the issues in my uploaded document."
  3. **Translate** (Purple theme)
     - Icon: Rephrase icon
     - Action: "Please translate my uploaded document to Hindi."
  4. **Arguments** (Orange theme)
     - Icon: Scale icon
     - Action: "Please generate arguments and counter-arguments for the legal position in my case."

### Welcome Message (Initial State)
Displayed when no messages exist:
- **Assistant Icon**: Large 48px assistant icon
- **Title**: "Welcome to LexAI"
- **Description**: "Your AI-powered Indian legal assistant. Ask any legal question or upload a document for analysis."
- **Quick Questions Section**: 6 suggested prompt buttons in a responsive grid:
  - "Summarize the key points of my uploaded document"
  - "Translate this document to Hindi"
  - "Generate arguments and counter-arguments for this case"
  - "Provide legal citations for this document"
  - "What are the key elements of a valid contract under Indian law?"
  - "Explain the process for filing a consumer complaint in India"

### Chat Messages Area
- **Scrollable container** with custom scrollbar
- **Message bubbles** with different styling for user vs assistant:
  - **User messages**: Right-aligned, blue background, white text
  - **Assistant messages**: Left-aligned, muted background, dark text
- **Avatar system**: User and assistant avatars
- **Markdown rendering**: Full markdown support for AI responses
- **Timestamps**: Formatted time display for each message
- **Loading indicator**: Animated sparkles icon when AI is processing

### File Upload Section
- **Upload indicator**: Shows when document is uploaded
- **File information display**:
  - Document name with file icon
  - Document type identification
  - Key information tags (up to 2 visible, with "+X more" indicator)
  - Processing status with loading animation
- **Remove file button**: X icon to clear uploaded document

### Input Form
- **Upload button**: Paperclip icon, triggers file selection
- **Text input**: "Ask a legal question..." placeholder
- **Send button**: Send icon, disabled when empty or loading
- **File input**: Hidden input accepting .txt, .md, .json, .pdf files

## Sidebar (Right Column)

### Calendar Widget
- **Title**: "Upcoming Events"
- **Content**: Shows next 3 Google Calendar events
- **Event display**: Date, title, and time
- **States**: Loading, error, empty, and populated
- **Link**: "View Full Calendar" to main calendar page

### Quick Actions
- **Title**: "Quick Navigation"
- **Two action buttons**:
  1. **Draft a Document**: Navigates to /draft
  2. **Show My Cases**: Navigates to /vault
- **Help text**: Information about LexAI workflows availability
